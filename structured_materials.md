  - Экономические системы (традиционная, централизованная, рыночная) и ключевые вопросы «что/как/для кого производить».

  - Показатели развития (ВВП/ВНП, структура экономики, уровень жизни).
- **Формулы/отношения**:
  - Альтернативные издержки = ценность наилучшей упущенной альтернативы (качественное определение, без числовой формулы).
  - Кривая производственных возможностей (КПВ) задает зависимость `Q_B = f(Q_A)` при полном использовании ресурсов (точные выражения зависят от выбранных благ).
  - Экономический рост измеряется темпом изменения реального ВВП: `g = (ВВП_t − ВВП_{t-1}) / ВВП_{t-1}`.
- **Практика/домашние задания**:
  - Доклад по плану (ресурс `resh.edu.ru`).
  - Таблица национальных экономических моделей (отличия, плюсы/минусы).
  - Рекомендованная литература и научные статьи для углубления.

## 2. `Лекция 2_ эластичность спроса и предложения`
- **Тема**: измерение чувствительности спроса и предложения.
- **Основные блоки**:
  - Ценовая эластичность спроса: виды (абсолютно неэластичный → абсолютно эластичный), точечная и дуговая эластичность.
  - Связь эластичности и выручки (`TR`) при изменении цены.
  - Эластичность спроса по доходу (товары низкого качества, нормальные, роскошь).
  - Перекрестная эластичность (субституты, комплементы, независимые товары).
  - Факторы эластичности спроса (заменители, время, доля в бюджете, насыщенность рынка и др.).
  - Ценовая эластичность предложения и ее факторы (время, специфика производства, хранение).
- **Формулы**:
  - Ценовая эластичность спроса: `E_d = (ΔQ / Q) / (ΔP / P) = (dQ/dP) · (P/Q)`.
  - Дуговая эластичность: `E_d = [(Q2 − Q1) / ((Q1 + Q2)/2)] / [(P2 − P1) / ((P1 + P2)/2)]`.
  - Эластичность спроса по доходу: `E_I = (ΔQ / Q) / (ΔI / I)`.
  - Перекрестная эластичность: `E_xy = (ΔQ_x / Q_x) / (ΔP_y / P_y)`.
  - Эластичность предложения: `E_s = (ΔQ_s / Q_s) / (ΔP / P)`.
  - Функция из задач: `Q_D = 200 − 4P` (линейный спрос).
  - Общая выручка: `TR = P · Q`; поведение `TR` зависит от `E_d`.
- **Практика**:
  - Тестовые вопросы о влиянии изменения цены на выручку при разных типах эластичности.
  - Расчет коэффициента эластичности предложения при изменении цены.
  - Задачи по функции спроса `Q_D = 200 − 4P`: точечная и дуговая эластичности, цена для заданной эластичности, максимальная выручка.
  - Комплексная задача на поиск равновесной цены и эластичностей спроса/предложения.

- **Формулы/примеры**:

- **Формулы и нормативы**:
  - Повременная оплата: `ЗП = Тарифная ставка × Отработанное время`.
  - Сдельная оплата: `ЗП = Расценка × Количество продукции`.
  - Сдельно-премиальная: `ЗП = (Расценка × Объем) + Премия`.
  - Аккордная (индив./бригадная): `ЗП_i = Фонд_аккорда × K_{ТУ_i} / ΣK_{ТУ}`.
  - Ограничение натуральной части выплаты: `Натуральная доля ≤ 20%`.
  - Пособие по временной нетрудоспособности: `П = min(Среднедневной заработок × %_стажа × Дни болезни, Максимум по ФЗ №255)`; `%_стажа = 60% / 80% / 100%`.
  - Компенсация за задержку выплат: `Компенсация = Сумма долга × (1/300) × Ставка ЦБ × Кол-во дней просрочки`.
- **Практика**:

### Свод формул (короткий справочник)
- **Эластичность и выручка**: `E_d = (ΔQ/Q)/(ΔP/P)`, `E_I = (ΔQ/Q)/(ΔI/I)`, `E_xy = (ΔQ_x/Q_x)/(ΔP_y/P_y)`, `E_s = (ΔQ_s/Q_s)/(ΔP/P)`, `TR = P·Q`.
- **Издержки**: `TC = FC + VC`, `AC = TC/Q = AFC + AVC`, `AFC = FC/Q`, `AVC = VC/Q`, `MC = ΔTC/ΔQ`, `TC (пример) = Q^2 + 2Q + 5`.
- **Амортизация**: `Н = 1/СПИ`, `A = (Ф_нач − Ф_ликв)/СПИ`, `A_месяц = A_год/раб. месяцев`, `A_ед = Ф_нач × (Q_факт/Q_рес)`, `A_t^{ум.ост} = Остаток_{t-1} × Н × k`, `A_t^{Σлет} = Ф_нач × (Ост. годы/Σ лет)`, `Σ лет = n(n+1)/2`.
- **Оборотные средства**: `K_ob = RP/ОС_сред`, `D_ob = Дни/K_ob`, `k_z = ОС_сред/RP`, `ΔОС_абс = ОСплан − ОСфакт`, `ΔОС_отн = (RP_факт/K_ob^{пред}) − ОС_факт`, `Норма = Среднесуточный расход × Дни`, `Норматив = Норма × Цена`.
- **Труд и оплата**: `t_н = t_о + t_в + t_об + t_п-з + t_отд + t_п.т`, `Н_выр = (T × K_p)/Н_вр`, `В = Q/T`, `Тр = T/Q`, `ЗП_поврем = Ставка × Время`, `ЗП_сдельн = Расценка × Объем`, `Пособие = Среднедневной × %_стажа × Дни (с учетом потолка)`, `Компенсация = Долг × (1/300) × Ставка ЦБ × Дни`.
